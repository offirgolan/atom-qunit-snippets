# QUnit Snippets for Atom

> [Atom](https://atom.io) Snippets for the [QUnit](https://qunitjs.com/) Javascript Testing framework.

## Install

```bash
$ apm install atom-qunit-snippets
```

Or Settings ➔ Packages ➔ Search for `atom-qunit-snippets`
